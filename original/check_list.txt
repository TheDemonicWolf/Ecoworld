1. Цель:
    С помощью агитационного материала (текст, изображения и прочее) донести до пользователей информацию о пагубном влиянии глобального потепления на окружающую среду, тем самым способствуя улучшению экологического состояния планеты.
2. Структура проекта:
    Научные статьи; +
    Видеолекция о способах предотвращения глобального потепления; +
    Калькулятор энергоэффективности; +
    Мастер-классы для детской аудитории.
3. Задачи тестирования:
    1. Функциональность:
        1. В работе кода не наблюдается никаких ошибок; +
        2. Отсутствуют неполадки с маршрутами; +
        3. Все ссылки работают стабильно. +
    2. Актуальность:
        1. Проект является востребованным; +
        2. Выполнена цель проекта.
    3. Пользовательский интерфейс:
        1. Пользователь не теряется в проекте; +
        2. Реализованы подсказки для пользователей; +
        3. Проект расчитан на аудиторию всех возрастов. +
    4. Безопасность:
        1. Соблюдается политика конфиденциальности;
        2. Связь проходит напрямую с разработчиками;
        3. Данные пользователя не распространяются за пределы диалога.
